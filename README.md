# Portfólio - Luis Henrique

## Sobre o projeto

Este projeto consiste no desenvolvimento de um portfólio pessoal online, criado como atividade prática da disciplina de Fundamentos da Programação Web.

O objetivo foi aplicar conceitos de desenvolvimento web utilizando HTML5, CSS3 e JavaScript, criando uma página única (Single Page Website) com navegação por âncoras.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Estrutura do projeto

├── index.html
│

│

└── script.js

## Funcionalidades

- Navegação por âncoras entre as seções da página;
- Layout responsivo para diferentes dispositivos;
- Seção de apresentação pessoal;
- Informações sobre formação acadêmica;
- Área de portfólio com projetos desenvolvidos;
- Formulário de contato com validação utilizando JavaScript.

## Seções do site

### Sobre mim
Apresentação pessoal, experiências profissionais e conhecimentos em desenvolvimento.

### Formação
Informações sobre minha formação acadêmica em Análise e Desenvolvimento de Sistemas e áreas de estudo.

### Portfólio
Projetos desenvolvidos envolvendo tecnologia, automação com Python e análise de dados utilizando Power BI.

### Contato
Informações para contato e formulário com validação.

## Projetos apresentados

### Automação de Cadastro com Python 🤖

Projeto desenvolvido utilizando Python para automatizar tarefas repetitivas de cadastro, buscando otimizar processos e reduzir atividades manuais.

### Dashboard de Vendas com Power BI 📊

Dashboard desenvolvido para análise de dados de vendas, apresentando indicadores e informações para auxiliar na tomada de decisões.
